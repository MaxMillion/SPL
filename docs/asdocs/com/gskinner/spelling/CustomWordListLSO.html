Classpublic class CustomWordListLSO
InheritanceCustomWordListLSO Object

CustomWordListLSO works with SpellingDictionary to support saving and loading custom word lists from Local Shared Objects. While useful on its own, it's also intended as a demonstration of how to implement different custom dictionary systems.
