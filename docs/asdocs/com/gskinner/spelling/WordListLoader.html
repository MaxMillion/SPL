Loads and parses word list files. It can handle either the GSPL file type, or a presorted (ascending, case-insensitive, and alphanumeric) plain text file containing a list of words separated by line breaks. It will automatically uncompress zlib-compressed binary text files â€“ this does not work with zip compression.

maximumParseTimeproperty 
public var maximumParseTime:uint = 50

Specifies the maximum number of milliseconds to spend per frame parsing word list data. This allows you to maintain application responsiveness while the parsing routine runs. For example, setting maximumParseTime to 20 would let animations continue to run smoothly, but increase the overall time required to parse the word list. Setting maximumParseTime to 3000 will cause the application to freeze completely for 2-3 seconds while the word list data is parsed, but the parsing will take less time in total.
