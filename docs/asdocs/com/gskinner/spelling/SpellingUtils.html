findIndex()method
public static function findIndex(list:Array, word:String):int

Uses a simple halving algorithm to find a word's position in a list of words. The list must be sorted ascending case-insensitive alphanumerically.

Parameters

list:Array — An array of strings sorted ascending case-insentive alphanumerically.
 
word:String — An array of strings sorted ascending case-insentive alphanumerically.

Returns
int — The word's index in the list, or the index it would be found at if it existed in the list.
mergeWordLists()method 
public static function mergeWordLists(list1:Array, list2:Array):Array

Merges two word lists. A word list is an array of strings sorted ascending case-insentive alphanumerically. This will not merge SpellingDictionary objects. Use this to merge loaded (or generated) word lists prior to using them in a SpellingDictionary.

Parameters

list1:Array — An array of strings sorted ascending case-insentive alphanumerically.
 
list2:Array — An array of strings sorted ascending case-insentive alphanumerically.

Returns
Array — the merged array.
