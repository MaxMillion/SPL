Classpublic class CharacterSet
InheritanceCharacterSet Object

CharacterSet provides data sets representing word characters. You can change these values at run-time to support different lanugages. This is primarily intended for use by TextHighlighter and SpellingDictionary.

innerWordCharsproperty
innerWordChars:String

A string containing all of the lower case characters that are considered to be part of a word, but must be contained within characters from the word chars set. For example: in "don't" the inner character "'" is considered part of the word, but in "'monkey'" it is not.
